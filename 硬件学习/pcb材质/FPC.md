# FPC
- 名称：柔性印刷电路板 flexible printed circuit board
- 材料：由PET/PI为基材 柔性电路中以绝缘薄膜、导体、粘接剂
- 性能：可以弯折 散热好 质量轻 充分利用立体空间
常见充分利用空间的方案： 卡槽＋[[介面卡]]  但是用FPC即可实现上面的功能，两个pcb板连接一个FPC

# R-FPC
- 名称：right flexible printed circuit board 刚性柔性印刷电路板（软硬结合板）
- 材料：FR-4 & PI
- advantage：
1. 结合PCB和FPC的优点 可以密集布线和高密度连接 
2. 可靠性
3. 机械性能：为电路板提供了良好的刚性和弹性。更好的抗振性 和 抗扭曲
4. 寿命更长：在恶劣条件下比一般电路板 更能保持良好性能
5. 节约空间
工艺：将pcb和fpc经过压合等工序

## FPC的基材
铜箔叠胶叠绝缘材料
#### 绝缘层:
放在导电层两侧涂上PI或者其他绝缘材料
作用：隔离导电层
PI：可以承受的温度范围:-200~300℃ 厚度：1/2mil 1mil 2mil
PET：更便宜 尺寸稳定性不好 耐温差 不适合[[smt]]和[[波峰焊]] 一般只用在插拔的连接排线（也在被PI取代）
#### 导电层：
材料:铜箔
常见厚度：1/3oz 1/2oz 1oz
#### 粘合层（胶层）
成分：环氧树脂
主要作用：固定导电层 提高绝缘强度和机械性能
厚度：13um 20um

# 带胶基材和无胶基材
通过特殊方法把绝缘层和导电层直接合成的材料
无胶更可靠 更小体积和质量 更容易加工 可靠性更高
有无毒无味抗菌等特殊性能要求的应用采用无胶fpc

# 压延铜&电解铜
铜基板材料常用 压延Cu和电解Cu
区别：

| |压延Cu | 电解Cu |
|---|---|---|
|制造工艺|铜板压制|电解机在阴极滚筒产生|
|结晶|片状组织|柱状|
|物理性质|略薄，弯曲性好，表面均匀度和平坦度较佳|粗中厚，纯度高|
|导电性|厚度更小，电阻更大|but导电率更高|

# 辅材
1. 保护膜 与基材相同的绝缘材料和胶结合的一种材料
- 结构：PI 交 离型纸
2. 补强材料：
- 作用：fpc局部为了焊接或者加强 支撑作用 提高局部的机械强度和稳定性
- 材质：
PI补强 用于插拔金手指 增强fpc的厚度和机械强度
FR-4 用于芯片和ic的支撑 提高抗弯折
金属补强 支撑 散热 可接地 平整度高
3. 其他辅助材料
电磁屏蔽膜 
纯胶膜
压敏胶（spa）（双面胶）
导电胶膜
PP

# 生成流程
单板：开料→烘烤→贴干膜 →曝光→ 显影 →蚀刻 → 脱膜→ 前处理 → 贴覆盖膜 → 压合 → 固化→表面处理→ 电测 → 装配 → 压合 → 固化→ 文字 → 外形→ 终检→包装 出货
双面板：开料→ 烘烤→ 钻孔→ 黑孔 → VCP→ 前处理→ 贴干膜 →曝光→ 显影 → 蚀刻 → 脱膜 → 前处理→ 贴覆盖膜 → 压合 → 固化→ 表面处理→ 电测→ 装配 → 压合 → 固化→ 文字 →外形→ 终检→包装 出货
